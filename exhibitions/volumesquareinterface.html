Volume Square Interface

Mobile hospital control system UI — monitoring negative-pressure safety, coordinating devices, and guiding role-based workflows under field constraints.

Overview

Volume Square Interface is a control and monitoring UI for rapidly deployed mobile hospitals. It integrates pressure states, airflow, alarms, device health, and bed occupancy into a single operational picture. The design emphasizes clear legibility in low-light tents, glove-friendly targets, and quick recovery from alarms with guided steps.
